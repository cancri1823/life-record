雲端設定

連接 Google 雲端硬碟以跨裝置同步資料
使用前置作業
1. 前往 Google Cloud Console 建立專案，啟用「Google Drive API」
2. 建立「OAuth 用戶端 ID」（類型：網頁應用程式），已授權的 JavaScript 來源填入您部署的網址（例如 https://your-id.github.io）
3. 將取得的 Client ID 貼在下方欄位並儲存
4. 點擊右上角「連接 Google 雲端」即可將資料與證書檔案同步到您自己的 Drive（資料存放於應用專屬區域，證書檔案存放於您 Drive 中的「人生檔案附件」資料夾）
資料管理
所有資料目前也備份在本機瀏覽器（localStorage）。若尚未連接雲端，清除瀏覽器資料將導致紀錄遺失，建議盡快設定雲端同步。
